openHAB - "memberdayakan rumah cerdas" - vendor dan agnostik teknologi open source otomatisasi rumah

The builds on F-Droid have FCM and crash reporting removed and therefore won't be able to receive push notifications from openHAB Cloud. Instead, notifications are polled on a regular basis in these builds.

openHAB, platform otomatisasi rumah open source berbasis Java yang mengintegrasikan dan menggabungkan banyak macam sistem rumah cerdas dan teknologi-teknologi ke dalam solusi tunggal.
On top of the unified abstraction layer all connected devices are available for the overarching automation rule engines and different user interfaces.

<b>Dukungan Produk</b>

Lebih dari 200 add-ons menyediakan dukungan untuk merek, perangkat, teknologi dan protokol komunikasi. Beberapa contoh seperti Z-Wave, Philips Hue, Amazon Echo, Chromecast dan Sonos. Temukan seluruh add-ons tersedia dan dukungan perangkat dan/atau fungsi-fungsi di: https://www.openhab.org/addons/
The app has an action plugin for Tasker and Locale included.

<b>Open Source Community</b>

 Forum dengan 13.000 lebih pengguna terdaftar adalah tempat untuk mendapat panduan, bantuan dan inspirasi. Gabung forum komunitas openHAB di https://community.openhab.org
Ketika menemukan sebuah permasalahan laporkan di dalam forum atau pada https://github.com/openhab/openhab-androis/issues
Jika berkenan menerjemahkan aplikasi di dalam bahasa Anda gabung bersama kami di https://crowdin.com/project/openhab-android

<b>openHAB Foundation</b>

Yayasan openHAB, sebuah organisasi nonprofit dengan misi mengedukasi masyarakat tentang kemungkinan-kemungkinan dan keuntungan atas solusi rumah cerdas yang bebas dan terbuka. Pelajari tentang misi dan layanan yayasan pada https://www.oprnhabfoundation.org
